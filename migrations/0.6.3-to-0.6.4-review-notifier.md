# PAM 0.6.3 to 0.6.4 — review notification relay

PAM 0.6.4 adds an authenticated, metadata-only review notification scanner and a durable relay for `review_required` decisions.

1. Keep the curator ledger and Fabric decision ACK outbox available to the scanner.
2. Run the scanner with one exact canonical scope; it emits no claim, provenance, subject, or secret material.
3. Store the Discord channel target in an owner-only file and use an already-authenticated OpenClaw gateway for delivery.
4. Run the relay from a timer. Its owner-only outbox and stable decision marker make retries recoverable without resending after an ambiguous success.

No existing PAM memory record changes shape in this release.
