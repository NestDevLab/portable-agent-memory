# 0.5.1 to 0.5.2 - Benchmark Threshold Gate

## Summary

PAM 0.5.2 adds a deterministic benchmark threshold gate for CI.

## Changes

- Adds `npm run benchmark:check`.
- Adds `tools/check-benchmark-thresholds.mjs`.
- Runs the benchmark threshold gate in GitHub Actions after the test suite.
- Documents the deterministic coverage thresholds and why real LLM runs remain
  optional instead of required for pull requests.

## Migration

No memory data migration is required. Existing graph-v1 workspaces remain
compatible.
