# Migration: PAM 0.3.1 to 0.4.0

PAM 0.4.0 expands the agent-layer guidance and supporting tooling while keeping
the existing graph format.

## Compatibility

- `memoryFormat` remains `graph-v1`.
- `graphSchemaVersion` remains `pam-graph-v1`.
- Existing 0.3.1 graph JSONL files remain valid.

## Required Actions

No automatic data migration is required.

Agents adopting 0.4.0 should review the updated runtime and agent-layer
guidance, then keep existing graph records unless their local workspace needs a
normal maintenance refresh.
